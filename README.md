This project involves building an interactive Sales Analytics Dashboard using Power BI and DAX to analyze sales performance, profit trends, customer behavior, and product performance across multiple years.

Key highlights:

Cleaned and modeled raw sales data in Power BI

Created DAX measures for key KPIs (Sales, Profit, Quantity, Discounts)

Applied time intelligence for year-wise trend analysis

Used DAX ranking to identify top and bottom performing products

Designed interactive visuals with slicers for dynamic insights

Outcome:
The dashboard provides actionable business insights and demonstrates strong skills in DAX, data modeling, and Power BI visualization.
